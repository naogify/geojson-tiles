# geojson-tiles

[geojson-vt](https://github.com/mapbox/geojson-vt) を使って、バイナリではないベクトルタイル（GeoJSONタイル）を作成するデモ。


## 使い方

セットアップ

```bash
$ git clone git@github.com:naogify/geojson-tiles.git
$ cd geojson-tiles/
$ npm install
```

以下を実行すると、`tiles` （xyz）ディレクトリに geojson ファイルを作成します。

```bash
$ npm run build
```
